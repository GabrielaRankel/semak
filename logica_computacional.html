1. O que são Fluxogramas? Simbologia Básica

Fluxogramas são diagramas que representam visualmente o fluxo de um processo ou algoritmo através de símbolos padronizados. Eles são ferramentas essenciais para planejar, documentar e comunicar processos de forma clara.

O que são algoritmos? Onde são usados?

São sequências de instruções claras, finitas e organizadas que dizem exatamente o que o computador deve fazer para resolver um problema. Na lógica, eles servem como o “passo a passo” que transforma entradas (inputs) em saídas (outputs).

Características de um Bom Algoritmo
Precisão: Passos claros e não ambíguos
Finitude: Deve terminar em algum momento
Entrada: Pode receber dados de entrada
Saída: Produz resultados
Efetividade: Cada passo deve ser executável

Quais os tipos de dados mais usados em algoritmos (portugol)?

1. Inteiro (int)
Valores numéricos sem casas decimais.
Ex: 10, -3, 0

2. Real (float)
Números com casas decimais.
Ex: 3.5, -0.8

3. Caractere (caracter / char)
Apenas um símbolo.
Ex: 'A', '@', '7'

4. Cadeia (string)
Conjunto de caracteres (texto).
Ex: "Olá", "Gabi"

5. Booleano (lógico)
Representa apenas verdadeiro ou falso.
Ex: verdadeiro, falso

6. Vetor (array)
Estrutura que armazena vários valores do mesmo tipo em posições.
Ex: vetor[1..5] de inteiro

Portugol é uma linguagem de pseudocódigo escrita em português, usada para aprender lógica de programação. Ele não é uma linguagem real de programação, mas sim uma forma simples e estruturada de representar algoritmos, facilitando o entendimento de comandos como entrada, saída, repetição e condições. Portugol Studio é um software (IDE) criado para escrever, testar e executar códigos em Portugol. Ele oferece editor de texto, destaque de sintaxe, execução dos algoritmos e mensagens de erro, ajudando estudantes a praticar lógica como se estivessem programando de verdade.

5. VEJA o que nossos ALUNOS PRODUZIRAM

Nesta parte, apresentamos alguns exemplos de códigos produzidos pelos alunos ao longo das aulas de Lógica de Programação. Foram desenvolvidos programas em Portugol utilizando funções, demonstrando como organizar melhor o código, reutilizar instruções e deixar os algoritmos mais claros.
