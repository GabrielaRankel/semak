1. O que são Fluxogramas? Simbologia Básica

Fluxogramas são diagramas que representam visualmente o fluxo de um processo ou algoritmo através de símbolos padronizados. Eles são ferramentas essenciais para planejar, documentar e comunicar processos de forma clara.

O que são algoritmos? Onde são usados?

São sequências de instruções claras, finitas e organizadas que dizem exatamente o que o computador deve fazer para resolver um problema. Na lógica, eles servem como o “passo a passo” que transforma entradas (inputs) em saídas (outputs).

O que é o sistema de arquivos? Como funciona em modo texto (cmd) e modo gráfico (explorer)?

O sistema de arquivos é o método que o sistema operacional usa para organizar, guardar e acessar os dados dentro de um dispositivo de armazenamento, como o HD, SSD ou pen drive. Ele define como os arquivos são nomeados, salvos e localizados, funcionando como uma grande “biblioteca” digital que mantém tudo em ordem para que o computador saiba onde está cada informação. Por exemplo: quando você cria um documento, o sistema de arquivos é quem guarda esse arquivo em uma pasta específica, registra seu nome, tamanho, data e tipo — e depois permite que você o abra novamente sempre que quiser.

No MODO TEXTO, como o Prompt de Comando (CMD) do Windows, o sistema de arquivos é acessado por comandos digitados.

Você usa instruções como:
dir → para listar os arquivos de uma pasta;
cd → para mudar de diretório (pasta);
copy ou del → para copiar ou apagar arquivos.
Tudo é feito com linhas de comando, e o usuário precisa saber os nomes exatos das pastas e arquivos.

MODO GRAFICO (Explorer): No modo gráfico, como o Windows Explorer, o sistema de arquivos é mostrado de forma visual, com ícones, janelas e pastas.

EX: Você usa o mouse para arrastar, copiar, apagar e abrir arquivos, sem precisar digitar nada. É o jeito mais intuitivo e prático, ideal para quem está começando.
