2. Conceitos Fundamentais

O que é uma Rede mínima?
Uma rede mínima consiste em pelo menos dois dispositivos conectados, podendo ser dois computadores diretamente ligados por um cabo ou via wireless.

O que é a Internet? O que a compõe?
A Internet é uma rede global de redes interconectadas, composta por:
Provedores de serviços (ISPs)
Backbones de fibra óptica
Data centers e servidores
Dispositivos de usuários finais
Roteadores e switches
Protocolos de comunicação (TCP/IP)

O que é a Web?
A World Wide Web (WWW) é um sistema de informação que opera sobre a Internet, utilizando o protocolo HTTP para acesso a documentos interligados por hyperlinks.

Um switch é um dispositivo crucial para conectar computadores, impressoras e outros dispositivos, permitindo que eles se comuniquem entre si. As entradas e saídas digitais no Arduino são fundamentais para a interação do microcontrolador com o mundo externo. As entradas digitais permitem que o Arduino receba sinais que possuem apenas dois estados possíveis, geralmente representados por alto (HIGH) ou baixo (LOW), correspondendo a tensões de 5V ou 0V, respectivamente. Isso é útil para detectar o acionamento de botões, sensores digitais ou interruptores. Já as saídas digitais permitem que o Arduino envie sinais nesses mesmos dois estados, controlando dispositivos como LEDs, relés ou pequenos motores. Combinando entradas e saídas digitais, é possível criar sistemas interativos, automatizados e inteligentes, tornando o Arduino uma ferramenta poderosa para projetos eletrônicos e prototipagem rápida.

7. Entradas e Saídas ANALÓGICAS no Arduíno

As entradas e saídas analógicas no Arduino permitem trabalhar com sinais contínuos, ou seja, valores que podem variar dentro de um intervalo específico, ao contrário das digitais, que têm apenas dois estados. As entradas analógicas utilizam o conversor analógico-digital (ADC) do Arduino para transformar sinais de tensão em valores numéricos que o microcontrolador consegue processar, geralmente variando de 0 a 1023 para tensões entre 0V e 5V. Isso é útil para ler sensores como potenciômetros, sensores de temperatura ou luminosidade, que retornam valores graduais. Já as saídas analógicas, na prática, são sinais de PWM (Pulse Width Modulation), que simulam variações de tensão permitindo controlar a intensidade de LEDs, a velocidade de motores ou a posição de servomotores. Assim, as entradas e saídas analógicas ampliam muito as possibilidades de controle e medição em projetos eletrônicos com Arduino.

10. Leitura de Sensor Analógico e Acionamento de Saída Digital

// Exemplo de código Arduino
int sensor = A0;
int led = 13;
void setup() {
  pinMode(led, OUTPUT);
  Serial.begin(9600);
}
void loop() {
  int valor = analogRead(sensor);
  if (valor > 500) digitalWrite(led, HIGH);
  else digitalWrite(led, LOW);
  Serial.println(valor);
  delay(100);
}

Este é um código Arduino simples que lê um valor de um sensor analógico (conectado ao pino A0) e controla um LED (conectado ao pino 13) baseado nesse valor.

11. Diodo, e Display de 7 Segmentos

O diodo permite a passagem de corrente em apenas um sentido. O diodo LED emite luz ao conduzir corrente. O display de 7 segmentos usa LEDs organizados para exibir números e caracteres simples.




Um display de 7 segmentos é um dispositivo eletrônico usado para mostrar números e algumas letras de forma simples. Ele é formado por 7 LEDs individuais (ou segmentos) dispostos em forma de “8”, que podem acender em combinações diferentes para formar dígitos

Tipos de displays
Anodo comum (Common Anode - CA):
Todos os anodos dos LEDs estão conectados juntos. Para acender um segmento, aplica-se LOW (0V) no segmento.
Catodo comum (Common Cathode - CC):
Todos os catodos estão conectados juntos. Para acender um segmento, aplica-se HIGH (5V) no segmento.

Funcionamento:
Cada segmento é controlado individualmente. Combinando quais segmentos acendem, é possível mostrar números de 0 a 9 e algumas letras (como A, b, C, d, E, F). Exemplo: Para formar o “8”, todos os segmentos são acesos; para o “1”, apenas os segmentos b e c acendem.
